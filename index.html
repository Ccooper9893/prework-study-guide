HTML

  • HTML, made of elements, is used to create the structure of a webpage.
    • Elements are types of content on the webpage
    • Types of elements: Text, Image, Header, and Lists.
  • The HEAD element contains information about the webpage.
    • The head opening element contains Metadata, which is not seen by users.
    • Inside the head element you can find these elements: title, base, link, and meta.
    • Information about contents of head elements
  • The Body element represents the visible content shown to the user.
    • The body consists of a header (h1, h2, etc.), main (bulk of the information), and footer.
    • Sections can be used to organize the information into groups
    • You can use (ul) to wrap (li) elements to make a new bullet point list!

Git

  • cd "file name" to change directory.
  • cd .. to move back to the previous directory
  • pwd to see current directory
  • ls to see files in current directory
  • git status: checks what branch we are currently on
  • git branch: shows all current branches
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout existing_branch: moves to branch specified
  • git clone URL: create a a local clone of the current repository
  • git add -A: adds all changed files to your working directory
  • git commit -m "comment": commits the files to our local repository (-m = messages)
  • git pull origin main: to check if local branch is up to date with base branch
  • git push origin feature/add-html: to push branch changes onto Github
  • touch file: to creature a new file (ex. touch script.js)
